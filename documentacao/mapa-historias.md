# Mapa de Histórias do Usuário

## 1. Organização do torneio
- Como organizador, quero criar um torneio para iniciar uma competição.
- Como organizador, quero definir o formato do torneio para estruturar a competição.
- Como organizador, quero gerar a estrutura da competição para visualizar grupos, tabela ou chaveamento.

## 2. Gestão de times e elenco
- Como organizador, quero cadastrar times no torneio para definir os participantes.
- Como organizador, quero cadastrar jogadores em um time para formar o elenco.
- Como organizador, quero associar um técnico a um time para identificar seu responsável.

## 3. Gestão de partidas
- Como organizador, quero cadastrar partidas no torneio para controlar os jogos.
- Como organizador, quero registrar o resultado de uma partida para atualizar a competição.
- Como organizador, quero visualizar a classificação ou o chaveamento para acompanhar o andamento do torneio.

## 4. Estatísticas
- Como organizador, quero registrar gols e assistências de uma partida para acompanhar o desempenho dos jogadores.
- Como organizador, quero registrar cartões amarelos e vermelhos para refletir eventos disciplinares.
- Como organizador, quero que o sistema calcule automaticamente a nota estatística dos jogadores para avaliar o desempenho individual.
- Como organizador, quero visualizar a artilharia do torneio para acompanhar os principais goleadores.
